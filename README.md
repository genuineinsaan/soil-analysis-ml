# 🌱 Soil Analysis Project

A smart AI-powered web application for soil detection and fertility analysis using Deep Learning and Machine Learning techniques.

This project analyzes a soil image uploaded by the user and predicts:

- Soil Type
- Fertility Level
- Moisture Level
- Soil Health Score
- Recommended Crops
- Soil Suggestions

The application also provides:

- Top 3 AI Predictions
- Confidence Scores
- Analysis Charts
- Downloadable PDF Reports

---

✅ Soil Image Classification using CNN + MobileNetV2
